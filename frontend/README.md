# Frontend

Aplicação React (Vite + TypeScript + Redux Toolkit) para:

- CRUD de produtos
- CRUD de matérias-primas
- CRUD de composição produto/matéria-prima
- Simulação de produção sugerida

## Scripts

```bash
npm run dev
npm run lint
npm run test
npm run build
npm run e2e
```
